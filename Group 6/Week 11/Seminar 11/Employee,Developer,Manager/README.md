# Задача: Система за управление на служители

**Разделете решението си на header (.h) и implementation (.cpp) файлове.**

---

## Клас `Employee` (Служител)

Базов клас, представящ служител в компания. Всеки служител се характеризира с:
- Име с произволна дължина
- Имейл адрес с произволна дължина
- Заплата

### Функционалности

- Конструктор с параметри. Класът няма конструктор по подразбиране.
- `getName()`, `getEmail()`, `getSalary()` — getter-и.
- `setSalary(amount)` — задава заплата. При отрицателна стойност — игнорира се.
- `print() const` — извежда информацията за служителя.
- Служителите да могат да се **сравняват** — по заплата.
- `operator==` — двама служители са равни ако имат еднакъв имейл.

---

## Клас `Developer` (Разработчик)

Наследява `Employee`. Добавя:
- Основен език за програмиране с произволна дължина
- Списък от владеени технологии с произволна дължина и произволен брой

### Функционалности

- Конструктор с параметри.
- `addTechnology(tech)` — добавя технология.
- `hasTechnology(tech) const` — връща дали разработчикът владее дадена технология.
- `print() const` — извежда информацията от базовия клас + език + технологии.

---

## Клас `Manager` (Мениджър)

Наследява `Employee`. Добавя:
- Отдел с произволна дължина
- Списък от имена на подчинените служители с произволна дължина и произволен брой

### Функционалности

- Конструктор с параметри.
- `addSubordinate(name)` — добавя служител под управление.
- `getTeamSize() const` — връща броя на подчинените.
- `print() const` — извежда информацията от базовия клас + отдел + подчинени.

---

## Пример за използване

```cpp
int main() {
    Employee e("Иван Иванов", "ivan@company.bg", 2500.0);
    e.print();

    Developer dev("Мария Петрова", "maria@company.bg", 3800.0, "C++");
    dev.addTechnology("STL");
    dev.addTechnology("CMake");
    dev.addTechnology("Git");
    dev.print();

    std::cout << dev.hasTechnology("Git")    << "\n";   // 1
    std::cout << dev.hasTechnology("Docker") << "\n";   // 0

    Manager mgr("Петър Георгиев", "peter@company.bg", 5200.0, "Engineering");
    mgr.addSubordinate("Иван Иванов");
    mgr.addSubordinate("Мария Петрова");
    mgr.print();

    std::cout << "Екип: " << mgr.getTeamSize() << " души\n";

    // Сравнение по заплата:
    std::cout << (dev < mgr) << "\n";   // 1 — 3800 < 5200

    // Copy — компилаторът генерира правилен copy constructor:
    Developer dev2 = dev;
    dev2.addTechnology("Docker");
    dev2.print();   // dev2 има Docker, dev — не
}
```
