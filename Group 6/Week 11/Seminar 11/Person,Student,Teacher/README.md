# Задача: Система за управление на университет

**Разделете решението си на header (.h) и implementation (.cpp) файлове.**

---

## Клас `Person` (Човек)

Базов клас, представящ човек. Всеки човек се характеризира с:
- Име с произволна дължина
- Имейл адрес с произволна дължина
- Възраст

### Функционалности

- Конструктор с параметри. Класът няма конструктор по подразбиране.
- `getName()`, `getEmail()`, `getAge()` — getter-и.
- `setAge(age)` — задава възраст. При стойност извън диапазона [0, 120] — игнорира се.
- `print() const` — извежда информацията за човека.
- Двама хора да могат да се **сравняват** — по възраст.
- `operator==` — двама хора са равни ако имат еднакъв имейл.


---

## Клас `Student` (Студент)

- Име с произволна дължина
- Имейл адрес с произволна дължина
- Възраст
- Факултетен номер
- Специалност с произволна дължина
- Списък от положени изпити — двойки (предмет, оценка) с произволна дължина и произволен брой

Класът **не трябва да дефинира нито един специален метод**.

### Функционалности

- Конструктор с параметри.
- `addExam(subject, grade)` — добавя положен изпит. При оценка извън диапазона [2.0, 6.0] — игнорира се.
- `getAverage() const` — връща средния успех. При липса на изпити — връща 0.
- `hasPassed(subject) const` — връща дали студентът е положил успешно даден предмет (оценка >= 3.0).
- `print() const` — извежда информацията от базовия клас + факултетен номер + специалност + изпити.

## Клас `Teacher` (Преподавател)

- Име с произволна дължина
- Имейл адрес с произволна дължина
- Възраст
- Катедра с произволна дължина
- Списък от предмети, които преподава — с произволна дължина и произволен брой
- Научна степен с произволна дължина (например „д-р", „проф.", „доц.")


### Функционалности

- Конструктор с параметри.
- `addSubject(subject)` — добавя предмет.
- `teaches(subject) const` — връща дали преподавателят води даден предмет.
- `print() const` — извежда информацията от базовия клас + научна степен + катедра + предмети.

## Пример за използване

```cpp
int main() {
    Person p("Иван Иванов", "ivan@uni.bg", 20);
    p.print();

    Student s("Мария Петрова", "maria@uni.bg", 21, "8MI0600512", "Информатика");
    s.addExam("ООП",        5.50);
    s.addExam("Математика", 4.75);
    s.addExam("Алгоритми",  6.00);
    s.addExam("Физика",     7.50);   // игнорира се — извън диапазон
    s.print();

    std::cout << "Среден успех: " << s.getAverage()       << "\n";
    std::cout << "Положил ООП:  " << s.hasPassed("ООП")   << "\n";  // 1
    std::cout << "Положил БД:   " << s.hasPassed("БД")    << "\n";  // 0

    Teacher t("Петър Георгиев", "peter@uni.bg", 45, "проф.", "Компютърни системи");
    t.addSubject("ООП");
    t.addSubject("Алгоритми");
    t.addSubject("Операционни системи");
    t.print();

    std::cout << "Води ООП:     " << t.teaches("ООП")     << "\n";  // 1
    std::cout << "Води Физика:  " << t.teaches("Физика")  << "\n";  // 0

    // Сравнение по възраст:
    std::cout << (s < t) << "\n";   // 1 — 21 < 45

    // Copy — компилаторът генерира правилен copy constructor:
    Student s2 = s;
    s2.addExam("БД", 5.00);
    std::cout << "\nОригинал среден успех: " << s.getAverage()  << "\n";
    std::cout << "Копие среден успех:    " << s2.getAverage() << "\n";
}
```
