# Задача 1
Създайте опростена файлова система, като:

Напишете интерфейс Addable с функция void add(string toAdd).

Напишете интерфейс Nullable с функция void null().

Напишете абстрактен клас MyFile, който е абстракция на файл, със следните член-данни и член-функции:

#### полета:
- string data - данните във файла

#### член-функции:
- сетър и гетър за data
- void printDetails() const - чисто виртуална функция, която принтира детайли за файловете, които зависят от вида на файла.
- не забравяйте виртуалния деструктор!

Напишете клас StandardFile, който наследява MyFile и Addable и Nullable, като:
- printDetails() да извежда "Standard file - data can be added and deleted"
- add(toAdd) добавя стрингът toAdd на края на data
- null() изтрива съдържанието на data

Напишете клас ConstantFile, който наследява MyFile и:
- задължително има конструктор с параметър data
- data да не може да се променя след създаването на инстанцията - помислете за два начина, по които можете да направите това
- printDetails() да извежда "Constant file - cannot be changed"

Напишете клас PermanentFile, който наследява MyFile и Addable, като:
- съдържанието на data не може да е празен стринг (задължително конструктор с параметри)
- printDetails() да извежда "Permanent file - data cannot be deleted"
- add(toAdd) добавя стрингът toAdd в началото на data

Напишете клас FileSystem, който представлява файловата система, който да има vector<unique_ptr<MyFile>> fileSystem - файловата система. Класът да може да изпринтира 
информация за всичките си файлове, да добавя файл, да изтрива файл. 

Изтествайте класовете.

# Задача 3

Напишете темплейтен клас Box, който е репрезентация на кутия, със следните полета и член-функции:

#### полета:
- vector<T> items - предметите в кутията

#### член-функции:
- конструктор по подразбиране
- функция за добавяне на предмет в кутията - по подразбиране предметът се добавя на края на вектора
- функция за махане на предмет от кутията по индекс

Нека T да има concept, че трябва да може да бъде сравняван (да има дефиниран оператор <=>).

Напишете темплейтен клас SortingBox, който наследява Box. Всеки път, щом елемент се добави в SortingBox, векторът items се сортира във възходящ ред.
