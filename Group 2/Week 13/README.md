# Семинар 13

На лекции 13 е взимано:
- Полиморфизъм.
- Ключова дума virtual.
- Статично и динамично свързване.
- Виртуални функции.
- Ключови думи - override, final.
- Виртуални таблици.
- Полиморфизъм - runtime and compile time.

---

## Задачи

1. Създайте система за управление на графични обекти. Системата съдържа следните класове:
  - Базов клас `GraphicObject`, който има:
    - поле id
    - поле name
    - поле zOrder
    - метод `virtual void draw()` - отпечатва базовата информация за обекта на стандартния изход.
    - метод `virtual std::unique_ptr<GraphicObject> clone()` - връща копие на обекта.
  - Производни класове (класове наследници) `Circle`, `Rectangle` и `Text`, които:
    - предефинират `draw()` и `clone()`.
    - маркират `clone()` със `final`.
  - Клас 'Render', който:
    - съдържа колекция от уникални указатели от тип `GraphicObject`.
    - има метод `draw()`, който вика `draw()` на всички `GraphicObject` от колекцията, така че zOrder да е от най-нисък към най-висок.
